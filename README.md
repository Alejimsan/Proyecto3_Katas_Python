# Proyecto3_Katas_Python

# Proyecto 3: Katas de Python - [Alejandro Jiménez]

Este repositorio contiene la resolución de 40 ejercicios (Katas) de programación en Python, desarrollados como parte del Proyecto 3. El objetivo es demostrar el dominio de los fundamentos del lenguaje, desde la manipulación de datos básicos hasta la Programación Orientada a Objetos (POO).

## Estructura del Repositorio

- `katas_python.py`: Archivo principal con la resolución de los 40 ejercicios. Cada uno incluye su enunciado y comentarios explicativos.
- `comentarios.txt`: Notas sobre el desarrollo, desafíos encontrados y lógica aplicada en los ejercicios más complejos.
- `README.md`: Descripción general del proyecto (este archivo).

## Contenidos Técnicos

A lo largo de las 40 katas, se han aplicado los siguientes conocimientos:

- **Tipos de datos y métodos**: Uso de strings, integers, floats, listas, tuplas y diccionarios.
- **Lógica de Control**: Estructuras condicionales (`if`, `elif`, `else`) y bucles (`for`, `while`).
- **Programación Funcional**: Implementación de funciones `map()`, `filter()`, `reduce()` y funciones `lambda`.
- **POO (Programación Orientada a Objetos)**: Creación de clases, constructores (`__init__`), métodos de instancia y gestión de atributos.
- **Gestión de Errores**: Manejo de excepciones con bloques `try`, `except` y lanzamiento de errores con `raise`.
- **Módulos**: Uso de librerías estándar como `math` y `functools`.
